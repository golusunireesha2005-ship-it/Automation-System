About AuraTask

Redefining personal task management using browser automation and natural language parsing.

Our Vision

AuraTask was built to bridge the gap between simple calendar apps and complicated project management software. We believe tracking your time and schedule should require minimal manual overhead. By integrating smart NLP (Natural Language Processing) right in the UI, tasks configure themselves from single sentences.

Technological Stack

  • Client Shell: Semantic HTML5 structure with modern, CSS custom properties.
  • Styling engine: Clean Vanilla CSS layout systems (Grid/Flexbox) with a unified glassmorphism theme, dark mode utilities, and keyframe animations.
  • Application logic: Clean ES6 Javascript, utilizing LocalStorage for fast data caching and secure user management.
  • Engine notifications: HTML5 Web Notification API paired with synthetic sound notification system.

Application Pipeline

1. Natural Input
2. AI Parse Engine
3. Storage & State
4. Real-time Alarm

Frequently Asked Questions

How does the AI Smart Input Parser work?

Simply write details about your task in one natural phrase in the AI Smart Input box on the dashboard. Our engine searches for key indicators such as priorities ("high", "medium", "low"), days ("today", "tomorrow", "Monday", "Friday"), and time notations ("9 AM", "16:30", "at 4 PM"). The task is then structured and saved instantly!

Will my task reminders function if the tab is in the background?

Yes! AuraTask's background clock runs continuously as long as the application page remains open in a browser tab. For optimal alerts, grant permission to "Show Notifications" when prompted, enabling system-wide desktop notifications.

Is my personal information stored securely?

Absolutely. AuraTask has a serverless design. All user authentication details, hashes, and task objects are written directly to your web browser's local sandbox storage (`localStorage`). No network calls are made, meaning absolute offline privacy.
